About P(Doom) Global Dashboard

The P(Doom) Dashboard is an open-source project to track, visualize, and share key metrics related to global AI safety, governance, and capabilities. It is designed to be transparent, extensible, and community-driven.

How It Works

  • Data is automatically synced daily from the pdoom-data repository.
  • Configurable filters control what data gets included.
  • Metrics are processed through a modular data pipeline for quality and consistency.
  • The dashboard displays live and historical data for public review.

Automated Data Sync

The dashboard automatically pulls data from the pdoom-data repository every day at 6:00 AM UTC. You can configure what data gets synced by editing config/data_sync.yaml.

Contributing Data

Anyone can contribute new data by adding files to the pdoom-data repository. Please prepare your data in the following format: 

{
  "metrics": [
    {"name": "safety_researchers", "value": 847},
    {"name": "governance_researchers", "value": 312}
    // ... more metrics ...
  ],
  "source": "your_source_name"
}

Submit your data via a pull request to the pdoom-data repository. Data will be automatically synced to the dashboard within 24 hours.
